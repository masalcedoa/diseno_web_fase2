Objetivo General

Mostrar los pasos en la programación de una página web conectada a base de datos y los elementos necesarios para su vinculación utilizando PHP.

Objetivos del OVI

-- Identificar los componentes PHP necesarios para vincular una página web a base de datos.

-- Implementar una solución básica para la explotación de datos.

-- Realizar resumen sobre el uso de PHP en una pagina web como mecanismo de acceso a una base de datos.
